Using MOIRA Data Lab

The Data

MOIRA contains the underlying cause of death code using International Classification of Diseases (ICD) codes for all persons who died in the US between 1950 and 2017 (limited to deaths from malignant neoplasms for the 1950-1961 time period).

Selection Criteria

For general queries, you can group results by Cause of Death, Time, Geography or Demographics and download the tabular data in Excel-compatible CSV format.

Visualizations

Plot mortality data by Race and Sex over time or generate maps to compare different geographic regions for a given time period. For additional visualization examples, see API Docs.

Data Query

Choose from all selection criteria to generate mortality rates as tabular data. Downloads are available in CSV or XLSX format.
Demographics

Plot mortality rates by Race and Sex over time to analyze health disparities among available demographic groups.
GeoAnalytics

Compare US mortality rates at the State and County level for a given time period. Use Clustering Analysis to locate hotspots.

Standard 63 OCMAP categories are available for cause of death grouping. Additionally, the 113 MOIRA specific categories are available for datasets that employ only ICD-9 and/or ICD-10 codes.

Custom Cause of Death

Custom ICD categories can be selected by providing individual ICD codes. The ICD code revision is required to correspond to the selected time period.

Custom Time Periods

Custom time periods can be created for any set of years from 1950-2017.

National, state, and county level data is available. The geographic region can be individually selected or joined into custom aggregated regions.

NOTE:
  • Alaska data is not available at the county level before 1979.
  • Alaska and Hawaii data is not available before 1960.
  • New Jersey data is not available from 1962-1963.
    • Age Group

    12 or 13 age groups are available depending on the time period.* Also, custom aggregate age groups can be selected.
